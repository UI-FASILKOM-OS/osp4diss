# VIMRC

## Create/Edit  "**.vimrc**"

```
cat > $HOME/.vimrc << EOF
syntax off
EOF

cat .vimrc

```

```
cbkadal@cbkadal:~$ cat > $HOME/.vimrc << EOF
> syntax off
> EOF

cbkadal@cbkadal:~$ cat .vimrc
syntax off

```
